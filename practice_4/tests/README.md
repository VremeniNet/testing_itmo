# Тесты

Разложите тесты по видам, например:

```
tests/
├── unit/         модульные тесты pricing.py
├── mocks/        тесты сервисов с подменой зависимостей
├── api/          тесты HTTP API
└── property/     тесты на основе свойств
```

Помимо этого, в случае нахождения бага нужно составить баг-репорт.
