Падение тестов является ожидаемым, так как в исходном коде найден дефект.

### Найденный дефект

Файл:

```text
src/shipping.py
```

По спецификации для международной доставки весом от 1 до 20 кг стоимость должна считаться так:

```python
500 + 150 * (weight_kg - 1)
```

Фактически в коде используется:

```python
500 + 100 * (weight_kg - 1)
```

Из-за этого стоимость международной доставки для веса больше 1 кг получается меньше ожидаемой.

Пример для веса 5 кг:

```text
Ожидается: 500 + 150 * 4 = 1100
Фактически: 500 + 100 * 4 = 900
```
